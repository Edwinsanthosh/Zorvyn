Finance dashboard API documentation

This page documents the available routes, access rules, common request bodies, and query parameters.

Section 1

Auth APIs

POST /api/auth/register Register a new user

Creates a user and returns both access token and refresh token.

{
  "name": "Meena",
  "email": "meena@example.com",
  "password": "meena123",
  "role": "Viewer"
}
POST /api/auth/login Login user 
{
  "email": "nivetha@example.com",
  "password": "nivetha123"
}
POST /api/auth/refresh Refresh expired access token 
{
  "refreshToken": "your_refresh_token_here"
}
POST /api/auth/logout Clear refresh token for current session 
{
  "refreshToken": "your_refresh_token_here"
}

Section 2

User Management APIs

Admin only
GET /api/users List all users

Returns all users except passwords.

POST /api/users Create user by admin 
{
  "name": "Priya",
  "email": "priya@example.com",
  "password": "priya123",
  "role": "Analyst",
  "status": "active"
}
PATCH /api/users/:id/role Update role 
{
  "role": "Admin"
}
PATCH /api/users/:id/status Update status 
{
  "status": "inactive"
}

Section 3

Record APIs

Protected routes
GET /api/records List records with pagination and filters

Supported query params: `page`, `limit`, `category`, `type`, `date`, `fromDate`, `toDate`, `minAmount`, `maxAmount`.

/api/records?page=1&limit=5&type=income&fromDate=2026-04-01&toDate=2026-04-10
GET /api/records/:id Get one record

Viewer can only access their own record. Analyst and Admin can access broader data.

POST /api/records Create record

Allowed roles: Analyst, Admin

{
  "title": "April Revenue",
  "category": "Revenue",
  "amount": 45000,
  "type": "income",
  "date": "2026-04-04",
  "note": "Collected from branch"
}
PUT /api/records/:id Update record

Allowed roles: Analyst, Admin

DELETE /api/records/:id Soft delete record

Allowed role: Admin. Sets `isDeleted` to `true`.

PATCH /api/records/:id/restore Restore soft-deleted record

Allowed role: Admin. Sets `isDeleted` back to `false` and clears `deletedAt`.

Section 4

Dashboard APIs

GET /api/dashboard/summary Summary totals

Returns total income, total expense, total balance, total records, and category breakdown.

GET /api/dashboard/categories Category totals

Returns grouped income, expense, and total values per category.

GET /api/dashboard/recent-activity Recent activity feed

Optional query param: `limit`

GET /api/dashboard/trends Monthly or weekly trends

Optional query param: `type=monthly` or `type=weekly`
